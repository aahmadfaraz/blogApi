# blogApi
blogAPI using Node + Express + MongoDB

## Endpoints:

### User:
#### * signup
            --> For signup
#### * login
            --> For loggin In

### Blog:
#### * blog
            --> For viewing all blogs (or by ID if id provided)
#### * add
            --> For adding new blog
#### * update
            --> For updating blog by id
#### * delete
            --> For deleting a blog by id
#### * blog/user
            --> For getting blogs of a specific user by userId
